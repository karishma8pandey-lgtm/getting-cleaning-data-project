Dataset Description : The dataset contains accelerometer data from Samsung Galaxy S smartphones.

Variables:

1. subject: ID of the subject (1–30)
2. activity: Activity performed
3. measurement variables: Mean and standard deviation

Transformations:

1. Merged training and test datasets
2. Extracted mean and std measurements
3. Renamed variables
4. Converted activity labels
5. Created averaged dataset

Units: Normalized sensor values
