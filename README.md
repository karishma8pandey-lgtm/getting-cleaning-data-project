Getting and Cleaning Data Project

Overview : This project demonstrates how to clean and prepare data collected from smartphone accelerometers.

Files Included:

1. run_analysis.R
2. tidy_dataset.txt
3. CodeBook.md

 Steps Performed:

1. Merged training and test datasets
2. Extracted mean and standard deviation measurements
3. Applied descriptive activity names
4. Labeled variables clearly
5. Created a second tidy dataset with averages

How to Run:

Run the script using:
source("run_analysis.R")
